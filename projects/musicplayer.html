Transportation Dashboard Overview

  • Tools Used: Microsoft Excel
  • Github URL: Project Link

This transaction dashboard was created using Microsoft Excel to analyze public transportation and bus fleet performance metrics. The dashboard uses a dark-themed design with blue and teal accents to present key insights in a clear and visually engaging manner.

High-level metrics provide an overview of system performance, including total passengers (6,587), average riders per trip (33), busiest route (East-West), and least busy route (South Line).

Fleet utilization analysis highlights operational efficiency, showing under-utilized buses (23%, 19 buses), over-utilized buses (43%, 36 buses), and well-utilized buses (35%, 29 buses).

Temporal analysis explores ridership patterns across multiple time dimensions, including peak travel hours between 2:00 PM and 6:00 PM, with peak activity recorded at 20:57:00.

Year-over-year analysis reveals a significant decline in ridership from 2023 to 2024, with an 83.50% decrease, indicating potential areas for operational and service improvements.

Additional insights include weekly ridership trends, with Sunday recording the highest passenger volume and Thursday the lowest, as well as AM versus PM travel patterns showing higher ridership during evening hours.

Key observations from the analysis suggest opportunities to optimize route planning, fleet allocation, and service frequency to improve overall system efficiency and ridership.
