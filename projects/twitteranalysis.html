ATM Performance Dashboard

  • Tools Used: Power BI, Excel
  • Github URL: Project Link

Designed and developed a comprehensive ATM analytics dashboard using Power BI to provide deep insights into ATM transaction performance, cash utilization, and customer usage behavior across multiple locations.

Collected, cleaned, and transformed raw ATM transaction data using Power Query, ensuring data accuracy and consistency for reliable reporting. Built a structured data model with appropriate relationships to support efficient analysis and scalable dashboard performance.

Created interactive visualizations and dynamic KPIs to track key metrics such as total transactions, withdrawal volume, cash replenishment cycles, downtime frequency, and location-wise performance. Implemented filters and slicers to allow users to explore data by date, ATM location, transaction type, and usage trends.

Applied analytical techniques to identify usage patterns, peak transaction periods, and underperforming ATMs, enabling data-driven insights for operational optimization. These insights help stakeholders improve cash management strategies, reduce ATM downtime, and enhance customer experience.

Focused on clear data storytelling and intuitive dashboard design, ensuring that complex datasets are translated into actionable insights that support strategic and operational decision-making.
